## 钉钉插件 for uTools

> 这是给 Linux 的同学准备的

## 更新记录

## v1.3.1

### 新增

- 支持右键菜单
- 支持部分[表情]


## v1.0.3

### 优化

- 点击消息提醒,显示钉钉窗口

## v1.0.2

#### 新增
- 优化界面，能在 800 * 600 下正常使用
  
#### 修复
- 在 uTools v0.8.2-beta，消息提醒不工作
- 在 uTools v0.8.2-beta，不能打开连接
