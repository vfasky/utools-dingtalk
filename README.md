### 安装

> 由于 uTools 插件打包后， 所有的静态资源都应该在包内，否则会加载失败

所以，你只能以开发模式运行


- 下载源码
  
```sh
git clone https://github.com/vfasky/utools-dingtalk.git
```

- 进入目录

```sh
cd utools-dingtalk
```

- 复制 `plugin.json`
- 打开 uTools, 参照 https://u.tools/docs/developer/welcome.html#%E5%BC%80%E5%8F%91%E8%80%85%E4%B8%AD%E5%BF%83
